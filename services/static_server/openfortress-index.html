How to install OpenFortress [Linux]

1. Install subversion via your distro's package manager.

2. Install Team Fortress 2 and Source SDK Base 2013 Multiplayer.

TF2 is around 25gb, Source SDK is about 3.1gb.

3. Locate your sourcemods folder.

This is likely either ~/.local/share/Steam/steamapps, or ~/.steam/debian-installation/steamapps/sourcemods.

4. In the sourcemods dir, run svn checkout https://svn.openfortress.fun/svn/open_fortress/ open_fortress.

This should be around 10gb.

5. Restart steam.

6. Find Open Fortress in your steam library, and add these launch flags: -steam -secure.

7. Launch! Warning: the audio is likely very loud.

8. In game, open the console with ` and type connect of.genessee.zah.rocks.

If there are issues with the resolution/monitor, add these launch flags: -displayindex 1 -h 1440 -w 2560.

For more performance, try adding: -novid -nojoy -nosteamcontroller -reuse -nohltv.
